# Text emphasis
* 굵게 -bald : * or _ 2개 앞뒤로 넣기
  **JAVA** __JAVA__
* 기울임 -italic : * or _ 1개 앞뒤로 넣기
  *JAVA* _JAVA_
* 밑줄 -underline : < U>< /U> 앞뒤로 (띄어쓰기 제거)
  <U>JAVA</U>
* 취소선 -strikethrough : ~~ 앞뒤로 넣기
  ~~JAVA~~

# Horizontal Line
